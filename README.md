# U8336Tea Teabag

## How do I install these formulae?
`brew install u8336tea/teabag/<formula>`

Or `brew tap u8336tea/teabag` and then `brew install <formula>`.

Or install via URL (which will not receive updates):

```
brew install https://raw.githubusercontent.com/U8336Tea/homebrew-teabag/master/Formula/<formula>.rb
```

## Documentation
`brew help <command>` to get help for commands

## Commands
**brew autoremove:** Removes a formula and all its dependencies. This is like `rmtree`, but it does not depend on a formula dependency list. This comes at the cost of effeciency.
